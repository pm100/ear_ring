### Option 1: Expo Go (Easiest, no Mac needed)
1. Install **Expo Go** from the App Store on your iPad
2. Run `npm start` in this project directory
3. Scan the QR code with your iPad's camera (or the Expo Go app)

\> ⚠️ You're using `expo start --dev-client`, which means you may need a **custom dev client** (Option 2) if your app uses native modules beyond what Expo Go supports.

---

### Option 2: Custom Dev Client (requires a Mac with Xcode)
1. On a Mac, run: `npm run ios -- --device` to build and install directly to a connected iPad
2. Or build with EAS: `eas build --platform ios --profile development` and install the `.ipa` via TestFlight or direct install

---

### Option 3: iOS Simulator (Mac only)
Run `npm run ios` on a Mac — you can select an iPad simulator in Xcode.

---

**Quickest path:** Install Expo Go on the iPad → `npm start` → scan QR code. If you hit issues with native features (audio/haptics), you'll need the custom dev client route on a Mac.
